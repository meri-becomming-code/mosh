About This Project

The MCC Accessibility Toolkit was born from the observation that accessibility compliance often falls on individual faculty members who lack the time or training to do it manualy. It provides a free, open-source alternative to expensive enterprise software.

Core Features

  • HTML Remediation: Automatically fixes table headers, contrast issues, and heading structures in Canvas exports.
  • Link Validator: Finds broken links and unsafe targets.
  • File Converters: Converts .docx, .xlsx, and .pptx files into clean, accessible HTML.
  • AI Alt Text: Uses Google Gemini to generate descriptions for images.

Get Started

This tool is available for Windows and Mac/Linux users running Python.

View on GitHub Download ZIP

Installation (Non-Technical)

  1. Download and install Python 3.10+.
  2. Download the Toolkit ZIP (button above) and extract it.
  3. Open the folder, double-click install_dependencies.bat (Windows) or run the setup command in terminal.
  4. Double-click LAUNCH_TOOLKIT.bat to start the app.
